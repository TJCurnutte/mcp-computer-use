Everything an agent can touch

A secure, allow-listed tool surface that turns natural language into desktop actions.

Screenshots & OCR

Capture the full screen, a region, or a window. Extract text on screen so agents can see and click.

Mouse & Keyboard

Move, click, scroll, drag, type, press hotkeys, and hold modifier keys with pyautogui-level precision.

Shell & Processes

Run safe shell commands, start long-lived processes, and stream stdout/stderr back to the agent.

Files & Clipboard

Read, write, list, and delete files in allowed directories. Get and set the clipboard.

Windows & Apps

List windows, open apps, focus a window, and click UI text using OCR coordinates.

Safety First

Allow-listed shell commands, blocklists, confirmation prompts for destructive ops, and path sandboxing.

Install in seconds

Reflex runs natively on macOS and shows a simple icon in your menu bar. No electron, no browser tabs.

  1. 1

    Build or download the .app

    Clone the repo and run MacMenuBar/MCPMenuBar/build_app.sh, or grab a release from GitHub.

  2. 2

    Move to /Applications

    Drag Reflex.app to /Applications and open it.

  3. 3

    Grant permissions

    Approve Screen Recording and Accessibility when prompted. The app guides you through it.

Connect your MCP client

Reflex uses the stdio bridge to connect any MCP client to its TCP server. Add it to your config once and it just works.

{
  "mcpServers": {
    "mcp-computer-use": {
      "command": "/Users/you/mcp-computer-use/.venv/bin/python",
      "args": [
        "/Users/you/mcp-computer-use/MacMenuBar/bridge/mcp_bridge.py"
      ],
      "cwd": "/Users/you/mcp-computer-use"
    }
  }
}
